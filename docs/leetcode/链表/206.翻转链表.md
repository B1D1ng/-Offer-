#### 题目
```
反转一个单链表。

示例:

输入: 1->2->3->4->5->NULL
输出: 5->4->3->2->1->NULL
进阶:
你可以迭代或递归地反转链表。你能否用两种方法解决这道题？
```
#### 思路：三指针法，时间复杂度O(n)
```
class Solution {
    public ListNode reverseList(ListNode head) {
        if(head == null || head.next == null)return head;
        //此时至少有两个节点
        //三指针法
        ListNode tail = head;
        ListNode prev = head;
        ListNode mid = head.next;
        ListNode last = mid;
        do{
            last = mid.next;
            mid.next = prev;
            prev = mid;
            if(last != null){
                mid = last;
            } 
        }while(last != null);
        tail.next = null;
        return mid;
    }
}
```
